# 日報 — 2026-04-16

**`web/` に Astro 製静的サイト**を立ち上げ、公開 CSV から**掲載中／終了一覧**を表示するところまで実装した日。あわせて **開始月**・**カテゴリ（ニコニコ動画ジャンル相当）**での絞り込みと、<strong>ホスティング方針（Cloudflare Pages）</strong>のドキュメント化を行った。

## サマリ

| 項目 | 内容 |
|------|------|
| スタック | **`web/`** に **Astro**（静的出力）。データは **`PUBLIC_EVENTS_CSV_URL`** 経由の **公開 CSV**（ビルド時 `fetch`）。**axios 不使用**は継続。 |
| 一覧・データ | 掲載用シート列（`id`, `published_at`, … `category`）を **`web/src/lib/events.ts`** でパース。`published_at` 降順。ニコニコ **getthumbinfo → サムネ（`.L`）**。 |
| 開催終了 | JST 暦日で `end_date` 翌日から終了。**`/` は掲載中のみ**、<strong>`/ended/`</strong> に終了イベント。並びは `end_date` 降順、同一日は `published_at` 降順。 |
| 開始月 | **`/month/YYYY-MM/`**・**`/ended/month/YYYY-MM/`**。`start_date` の JST 暦月で絞り込み。**`MonthFilter`**（ピル型ナビ）。 |
| カテゴリ | **`web/src/lib/categories.ts`** にジャンル **17 件**（表示ラベルは CSV `category` と完全一致）。**`CategoryFilter`**（セレクト）。**`/category/{slug}/`**・**`/ended/category/{slug}/`**。 |
| dev の 404 対策 | 当初 `getStaticPaths` が「CSV に行がある slug のみ」だと **`astro dev` でキャッシュとずれて 404** になり得たため、<strong>定義済み 17 slug は常に静的パス生成</strong>（`allNiconicoCategorySlugs`）に変更。 |
| ドキュメント | **`doc/static-site-strategy.md`** を実装・運用に同期（§2.2〜2.3、§3 のカテゴリ行、改訂履歴など）。**デプロイ先決定**をコミット。 |
| 未実施（次回以降） | Cloudflare Pages 本番接続、`/terms/` 本文、投稿フォーム URL のナビ反映、カテゴリの複合 URL（任意）。 |

## 会話・検証メモ（補足）

| 内容 |
|------|
| カテゴリはニコニコ動画のジャンル一覧に合わせた **日本語ラベル**をシートに入れ、フロントは **slug（ASCII）** で URL を分ける。 |
| **`npm run build`** は CSV 取得込みで月別・カテゴリ別ページが生成される。**`astro dev`** は動的ルートの取り扱いが異なるため、カテゴリは **常時パス生成**で体験を整えた。 |
| **月フィルタとカテゴリは同一 URL を併用していない**（どちらか一方の一覧で絞り込み）。複合が必要なら別タスク。 |
