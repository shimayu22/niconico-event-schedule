## 主な作業内容（コミット順・新しい方が上）

1. **`0759a03` あと何日で開催か表示する**  
   - [jst-date.ts](../../web/src/lib/jst-date.ts) に `calendarDaysBetween` / `daysUntilStartFromToday` / `formatDaysUntilStartJa` を追加。  
   - [EventCard.astro](../../web/src/components/EventCard.astro) で `todayYmd` が渡されたときだけ「あと n 日」を表示（開催中・終了ページには出さない）。  
   - [EventGrid.astro](../../web/src/components/EventGrid.astro) 経由で `todayYmd` を渡す。  
   - 開催予定ページ 3 箇所（`upcoming/index` / `category/[slug]` / `month/[ym]`）で `todayYmd={todayYmd}` を指定。  
   - **注意:** 静的サイトのため日数は **ビルド時点の JST 日付**で固定される（日次 Deploy Hook で更新）。

2. **`e890464` 開催中イベントページの並び順を、開催が早い順に変更**  
   - `sortOngoingEventsForDisplay` を追加し、開催予定と共通の `sortByStartDateAscForDisplay` に委譲。  
   - 第 1 キー: `start_date` 昇順、第 2 キー: `published_at` 降順（同開始日時）。  
   - 対象: [index.astro](../../web/src/pages/index.astro)、[month/[ym].astro](../../web/src/pages/month/[ym].astro)、[category/[slug].astro](../../web/src/pages/category/[slug].astro)。

3. **`3252210` 開催予定イベントページの並び順を、開催が近い順に変更**  
   - `sortUpcomingEventsForDisplay` を追加（後に開催中と共通化）。  
   - CSV 全体のソート（`events.ts` の降順）はそのまま、**表示用に昇順で並べ替え**。  
   - 対象: [upcoming/index.astro](../../web/src/pages/upcoming/index.astro)、[upcoming/category/[slug].astro](../../web/src/pages/upcoming/category/[slug].astro)、[upcoming/month/[ym].astro](../../web/src/pages/upcoming/month/[ym].astro)。
